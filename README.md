# Sistem Informasi Rental Buku / Komik Digital (E-Library) 

# 1. Pendahuluan

Perkembangan teknologi digital telah mengubah cara masyarakat mengakses informasi dan hiburan, termasuk dalam membaca buku dan komik. Sistem perpustakaan konvensional memiliki keterbatasan dalam pencatatan data, pengelolaan stok, serta monitoring peminjaman. Oleh karena itu, dibutuhkan sebuah sistem digital yang mampu mengelola seluruh proses tersebut secara terstruktur dan efisien.

Sistem Informasi Rental Buku / Komik Digital (E-Library) adalah aplikasi berbasis web yang dirancang untuk mempermudah administrator dalam mengelola data buku, genre, penulis/penerbit, anggota, serta proses peminjaman dan pengembalian buku secara digital.

Aplikasi ini menggunakan arsitektur Decoupled Architecture, yaitu backend dan frontend dipisahkan sepenuhnya. Backend dibangun menggunakan CodeIgniter 4 REST API, sedangkan frontend menggunakan VueJS 3 SPA dengan TailwindCSS.
